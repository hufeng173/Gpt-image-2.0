# 太极图本地图片放置说明

请把你自己的图片放到当前目录：`public/brand/`

必须文件：

1. `background.png`
   - 用途：主页面水墨背景
   - 建议尺寸：1920 x 1080
   - 格式：PNG 或 JPG 均可；代码默认引用 PNG

2. `logo.png`
   - 用途：左侧顶部品牌 Logo
   - 建议尺寸：640 x 180，透明背景 PNG
   - 内容建议：太极图图标 + “太极图”文字；可包含 EastWill 小字

3. `seal.png`
   - 用途：顶部头像、英雄区印章装饰
   - 建议尺寸：256 x 256，透明背景 PNG
   - 内容建议：红色篆刻印章，例如“太极图”或“EastWill”

可选文件：

4. `avatar.png`
   - 用途：后续用户头像
   - 建议尺寸：256 x 256

注意：文件名必须完全一致，尤其是 `background.png`、`logo.png`、`seal.png`。
